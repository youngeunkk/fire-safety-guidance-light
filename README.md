## 🚀소방안전 유도등

연기 및 화재 감지 기능을 넣은 프로젝트입니다!

## 🏁사용된 센서

* 아두이노 보드, 브래드보드
* LED 🔴🟢🟡
* Flame Sensor (불꽃 감지 센서)
* MQ2 (연기 감지 센서)
* Button
* Buzzer
  
## 🌙주요 기능
* 연기 감지 (LED 🟡 ON )
* 화재 감지 (LED 🔴 ON )
* 연기 감지 및 화재 감지 시 Buzzer (경보음) 출력 
* 점검 버튼 클릭시 점검 중 LED 🟢 ON
